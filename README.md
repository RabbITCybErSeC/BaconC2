# BaconC2

Bacon is a small C2 framework, which aims to support stealthy communication techniques.

## Why?

As of now just to learn about different (stealthy) C2 techniques, and a bit of malware development.

## Goals

- Easy web-interface for managing the connected agents.
- C2 channel over unconventional using a secure channel
- Agent which supports features such as memory dumps, token stealing, and more.
- Easy pivoting through by featuring the option to chain agents
